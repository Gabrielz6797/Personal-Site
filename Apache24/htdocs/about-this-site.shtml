Acerca de este sitio

Este sitio fue creado para el curso CI-0137 - Desarrollo de Aplicaciones Web impartido por el profesor Braulio José Solano Rojas durante el primer ciclo lectivo del año 2023, con los objetivos de:

  • Comprender los fundamentos del desarrollo de sitios web estáticos.
  • Introducir en las aptitudes básicas el uso de (X)HTML 5, CSS y Javascript.
  • Realizar un sitio web adaptativo básico.
  • Reconocer las dificultades de la generación de contenido.
  • Reconocer las dificultades del diseño gráfico.
